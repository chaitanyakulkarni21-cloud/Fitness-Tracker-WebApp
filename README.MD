# 🏋️ Fitness Tracker App (MERN Stack)

## 📌 About the Project

This is a Fitness Tracker web application built using the MERN Stack (MongoDB, Express, React, Node.js).

The app helps users to track their daily workouts and manage their fitness activities.

---

## 🚀 Features

* 🔐 User Signup and Login
* 📊 Dashboard to view workout details
* ➕ Add new workouts
* 📅 Track workouts by date
* 🔄 Real-time data using API

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **API:** REST API
* **Styling:** CSS / Styled Components
